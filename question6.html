HTML Tags

  1. head Tag : Contains metadata/information for the document
  2. html Tag : Defines the root of an HTML document
  3. body Tag : Defines the document's body
  4. a Tag : Defines a hyperlink
  5. b Tag : Defines bold text
  6. ol Tag : Defines an ordered list
  7. table Tag : Defines a table
  8. ul Tag : Defines an unordered list
